## 🎯 What I Do

I run [**AK SEO Labs**](https://akseolabs.com) — a Taiwan-based SEO studio focused on:

- **Semantic SEO** & topical authority architecture
- **AI content pipelines** integrated into team publishing workflows
- **Programmatic SEO** for structured-data sites
- **Technical SEO** refactors, internationalization, Core Web Vitals
- **Brand matrix / PBN / GEO** optimization

Outside client work, I build open-source tools for the SEO and AI-coding communities, and publish anti–low-end-vendor education on [Threads](https://www.threads.com/@darkseoking) — drawn from 10+ years in the field and 300+ SEO writer interviews.
